Mettre à jour la version de Go

  1. Désinstaller la version actuelle de Go (Windows: applications installées)
  2. Installer la dernière version du .msi de Go : Page de téléchargement
  3. Dans le dossier où se trouve go.mod :
    1. "go mod tidy"
    2. Changer la version dans le go.mod
    3. "go mod tidy"
